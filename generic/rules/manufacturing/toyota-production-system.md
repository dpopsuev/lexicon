# Toyota Production System (TPS)

**Core idea:** Two pillars — Just-in-Time (make only what's needed) and Jidoka (stop on defect). Everything else supports these.

## Two Pillars

- **Just-in-Time (JIT)** — make only what is needed, when needed, in the amount needed
- **Jidoka** — automation with human judgment; stop the line when a problem is detected

## Supporting Concepts

| Concept | What |
|---|---|
| **Kaizen** | Continuous improvement — small changes, always |
| **Kanban** | Visual signal cards to trigger production |
| **Gemba** | Go and see — observe where work happens |
| **Andon** | Visual alert board — signal problems immediately |
| **Poka-yoke** | Error-proofing — prevent mistakes by design |
| **Heijunka** | Level the workload — prevent overburden and unevenness |
| **5S** | Sort, Set in order, Shine, Standardize, Sustain |

## Three Enemies (3M)

- **Muda** — waste (7 types)
- **Mura** — unevenness (inconsistency)
- **Muri** — overburden (stress on people/machines)
