# Scaled Agile Framework (SAFe)

**Core idea:** Apply agile principles at enterprise scale across multiple teams, programs, and portfolios.

## Levels

- **Team** — Scrum/Kanban teams (2-week iterations)
- **Program** — Agile Release Train (ART), 8-12 week Program Increments
- **Large Solution** — coordinates multiple ARTs
- **Portfolio** — strategy, investment, governance

## Key Concepts

- Program Increment (PI) Planning — face-to-face, cross-team alignment
- Architectural Runway — infrastructure to support upcoming features
- Inspect & Adapt — scaled retrospective
