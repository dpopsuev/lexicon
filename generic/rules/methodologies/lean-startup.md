# Lean Startup

**Core idea:** Treat every product idea as a hypothesis. Build the minimum viable product (MVP), measure real user behavior, learn whether to pivot or persevere.

## Build-Measure-Learn Loop

1. **Build** — create an MVP to test the hypothesis
2. **Measure** — collect data on user behavior
3. **Learn** — validate or invalidate the hypothesis
4. **Pivot or Persevere** — change direction or double down

## Key Concepts

- **MVP** — minimum viable product (smallest thing that tests the hypothesis)
- **Validated Learning** — learning backed by data, not opinion
- **Innovation Accounting** — metrics that matter, not vanity metrics
- **Pivot** — structured course correction
