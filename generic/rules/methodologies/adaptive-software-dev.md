# Adaptive Software Development (ASD)

**Core idea:** Software development is a complex adaptive system. Replace plan-design-build with speculate-collaborate-learn.

## Three Phases

1. **Speculate** — plan with the understanding that plans will change
2. **Collaborate** — work together across boundaries
3. **Learn** — review results, adapt the approach

## Key Insight

Traditional "plan" implies certainty. "Speculate" acknowledges uncertainty. The cycle is continuous — each learning phase feeds the next speculation.
